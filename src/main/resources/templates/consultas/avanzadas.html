# Tipo de Consulta Método Repository Descripción
1 Derived Query findByRolNombre() Buscar usuarios por nombre de rol
2 Derived Query findByEmailContainingOrNombreContainingOrApellidoContaining() Búsqueda parcial en email, nombre o apellido
3 Derived Query findByFechaCreacionBetween() Usuarios en rango de fechas
4 @Query countByActivo() Contar usuarios por estado
5 Derived Query findAllByOrderByFechaCreacionDesc() Ordenados por fecha descendente
6 @Query countUsersByRol() Estadísticas por rol
7 Derived Query findByActivo() Buscar por estado activo/inactivo
8 @Query findByRolNombreAndActivo() Combinación rol + estado
